# ProjectCars_Fan

Fancontroll with Project Cars Arduino Dashboard
And show data with a i2c LCD Panel

### Used Parts

- Arduino Nano
- BD137
- Red LED
- I2C 20x4 Display
- 12 V Fan
- Software form ![ws-clan.de](http://ws-clan.de/pc_dash/)
